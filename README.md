# 🧙‍♀️ una-marvel02-java

Projeto desenvolvido para prática de **estruturas condicionais em Java**, com foco no uso de `switch case`.

---

## 📚 Sobre o projeto

Este projeto simula diferentes fases da personagem Wanda (Marvel), utilizando estruturas de decisão para definir comportamentos com base em uma variável.

Foram implementadas duas abordagens:

* 🔹 `WandaSwitchAntigo.java` → Sintaxe tradicional do `switch`
* 🔹 `MarvelSwitch.java` → Nova sintaxe do `switch` (Java moderno)

---

## 🧠 Conceitos aplicados

* Estrutura `switch case`
* Uso do `break`
* Uso do `default`
* Comparação com `if-else`
* Organização e legibilidade de código

---

## ▶️ Como executar

1. Compile o arquivo:

```
javac NomeDoArquivo.java
```

2. Execute:

```
java NomeDoArquivo
```

📌 Exemplo:

```
javac WandaSwitchAntigo.java
java WandaSwitchAntigo
```
